# ศักยภาพการใช้ที่ดิน (ภาคใต้)

Interactive Leaflet web map of land-use suitability and consistency analysis for South Thailand.

## Run locally

```
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy on GitHub Pages

1. Push this folder's contents to a GitHub repo (root, or a `docs/` folder).
2. Repo Settings → Pages → set the source branch/folder.
3. The site will be published at `https://<username>.github.io/<repo>/`.
